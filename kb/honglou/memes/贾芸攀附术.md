# 贾芸攀附术

[[贾芸攀附术]]出自第二十四回，是[[贾芸]]从[[贾府]]旁支落魄子弟逆袭谋得[[大观园]]种树差事的多步层层递进法。此法以"攀附认亲。借资备礼。精准送礼。谋得差事"为核心链条，展现了底层人物在大家族中借势上位的社交智慧。

## 技能定义

| 字段 | 值 |
|------|-----|
| 名称 | 贾芸攀附术 |
| 出处 | [[第024回]] |
| 创始者 | [[贾芸]] |
| 类型 | 社交·攀附·谋差 |
| 核心原则 | 层层递进、借势而为、精准出击 |
| 适用对象 | 在大家族/大组织中谋求机会的底层边缘人 |
| 复杂度 | 低 |

## 适用条件

| 条件 | 说明 |
|------|------|
| 适用场景 | 需在等级森严的大家族或大组织中从底层谋取差事/机会 |
| 前提条件 | 有可攀附的对象（家[[族长]]辈、掌权人物）、有可筹措的启动资源 |
| 风险因素 | 攀附对象不肯认亲、中间人不可靠、资金链断裂 |
| 所需资源 | 少量启动资金（或仗义资助者）、一件拿得出手的礼物、得体的言辞 |
| 成功标志 | 获得差事/职位，进入目标系统 |

## 操作步骤

| 步骤 | 名称 | 操作说明 | 原文体现 |
|------|------|----------|----------|
| 1 | 攀附认亲 | 在关键人物面前主动示好，抓住一切机会拉近关系 | [[贾芸]]偶遇[[贾宝玉|宝玉]]时[[贾宝玉|宝玉]]戏称"倒像我的儿子"，[[贾芸]]立即顺水推舟认之为父，伶俐答道"[[摇车里的爷爷拄拐的孙孙|摇车里的爷爷，拄拐的孙孙]]"（024-004） |
| 2 | 借资备礼 | 多方筹措启动资金，备好投其所好的见面礼 | [[贾芸]]向母舅[[卜世仁]]赊借[[冰片]][[麝香]]遭拒（024-007），路遇[[倪二|醉金刚]][[倪二]]仗义资助十四两银子（024-010），次日便买了冰麝（024-013） |
| 3 | 精准送礼 | 选择最佳时机、以最得体的方式将礼物送到掌权者手中 | [[贾芸]]在[[王熙凤|凤姐]]出门时拦住[[请安]]，谎称"朋友开香铺送了[[冰片]][[麝香]]"，转送[[王熙凤|凤姐]]作为[[端阳佳节|端午节]]礼之用（024-014），[[王熙凤|凤姐]]"又是得意又是欢喜" |
| 4 | 谋得差事 | 趁掌权者心情大好之时，顺理成章提出请求 | [[王熙凤|凤姐]]收了礼次日，[[贾芸]]来领银子，[[王熙凤|凤姐]]爽快安排种树差事："你到午错的时候来领银子，后儿就进去种树"（024-019） |

## 核心原则

| 原则 | 说明 | 原文例证 |
|------|------|----------|
| 顺势而为 | 不强行攀附，而是在对方主动递话时顺势接住 | [[贾宝玉|宝玉]]随口说"倒像我的儿子"，[[贾芸]]立刻接口认父（024-003）（024-004） |
| 借力打力 | 启动资金不求权贵，找仗义的底层豪侠相助 | [[倪二]]虽是泼皮，"却因人而使，颇颇的有义侠之名"（024-011），主动借银十四两 |
| 投其所好 | 礼物要送到关键人物的心坎上 | [[王熙凤|凤姐]]正要办[[端阳|端阳节]]礼，"采买香料药饵的时节"，[[贾芸]]的冰麝正是及时雨（024-015） |
| 低调示弱 | 以弱者姿态出现，降低对方的戒备和嫉妒 | [[贾芸]]在[[王熙凤|凤姐]]面前自称"母亲怕我年轻"、"婶子休要心焦"（024-014），姿态谦卑 |

## 核心原文

> 原来这[[贾芸]]最伶俐乖觉，听[[宝玉]]这样说，便笑道：「俗语说的，`摇车里的爷爷，拄拐的孙孙'。虽然岁数大，山高高不过太阳。只从我父亲没了，这几年也无人照管教导。如若宝叔不嫌侄儿蠢笨，认作儿子，就是我的造化了。」 （024-004）

> [[贾芸]]出了[[荣国府]]回家，一路思量，想出一个主意来，便一径往他母舅[[卜世仁]]家来。……[[贾芸]]笑道：「舅舅说的倒干净。我父亲没的时候，我年纪又小，不知事。后来听见我母亲说，都还亏舅舅们在我们家出主意，料理的丧事。难道舅舅就不知道的，还是有一亩地两间房子，如今在我手里花了不成？……」 （024-007）（024-008）

> [[贾芸]]道：「老二，你且别气，听我告诉你这原故。」说著，便把[[卜世仁]]一段事告诉了[[倪二]]。[[倪二]]听了大怒…… （024-010）

> 那天已是掌灯时候，[[贾芸]]吃了饭收拾歇息，一宿无话。次日一早起来，洗了脸，便出南门，大香舖里买了冰麝，便往[[荣国府]]来。 （024-013）

> [[贾芸]]道：「有个原故，只因我有个朋友，家里有几个钱，现开香舖。只因他身上捐著个通判，前儿选了云南不知那一处，连家眷一齐去，把这香舖也不在这里开了。便把帐物攒了一攒，该给人的给人，该贱发的贱发了，象这细贵的货，都分著送与亲朋。他就一共送了我些冰片，麝香。我就和我母亲商量，若要转卖……」 （024-014）

> [[凤姐]]正是要办端阳的节礼，采买香料药饵的时节，忽见[[贾芸]]如此一来，听这一篇话，心下又是得意又是欢喜，便命[[丰儿]]：「接过芸哥儿的来……」 （024-015）

> [[凤姐]]道：「……你到午错的时候来领银子，后儿就进去种树。」说毕，令人驾起香车，一径去了。 （024-019）

## 应用参考

| 场景 | 建议调整 |
|------|----------|
| 现代职场求职位 | 认亲步骤改为在行业活动/社交场合自然搭上线，借资改为众筹或小成本制作作品集 |
| 体制内谋调动 | 认亲和送礼步骤需更隐蔽，重点在投其所好和时机把握 |
| 创业拉投资 | 借资步骤对应种子轮融资，精准送礼对应定制化商业计划书 |

## 补充说明

补充说明：贾芸攀附术的成功前提是他确实有办事能力。他虽走捷径入门，但种树差事终究要靠实干完成。送礼只能敲开门，能不能站稳终究看真本事。
