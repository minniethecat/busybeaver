# 协理宁国府法

[[王熙凤]]受[[贾珍]]之托，协理[[宁国府]][[秦可卿]]丧事，在短时间内将混乱局面整顿得井井有条。此法体现了从诊断到执行的一整套危机管理方法论。

## 技能定义

| 字段 | 值 |
|------|-----|
| 名称 | 协理宁国府法 |
| 出处 | [[第013回]]、[[第014回]] |
| 创始者 | [[王熙凤]] |
| 类型 | 管理·组织整顿 |
| 核心原则 | 先诊断、再分工、后威信、继以执法 |
| 适用级别 | 中层管理者接手混乱组织时 |
| 复杂度 | 中等 |

## 适用条件

| 条件 | 说明 |
|------|------|
| 适用场景 | 组织陷入管理混乱、职责不清、效率低下、营私舞弊时 |
| 前提条件 | 拥有充分授权（"[[对牌]]"即尚方宝剑） |
| 风险因素 | 严刑峻法可能树敌；需有足够权威背书；需全身心投入 |
| 所需资源 | 明确的人事花名册、奖惩手段、时间精力 |
| 成功标志 | 令行禁止，人人各司其职，混乱消失 |

## 操作步骤

| 步骤 | 名称 | 操作说明 | 原文体现 |
|------|------|----------|----------|
| 1 | 先理头绪 | 接手后不急于开始，先坐下来梳理问题，摸清脉络 | "太太只管请回去，我须得先理出一个头绪来，才回去得呢"（013-022） |
| 2 | 诊断五大弊病 | 分析组织现存的核心顽疾 | "头一件是人口混杂，遗失东西；第二件，事无专执，临期推委；第三件，需用过费，滥支冒领；第四件，任无大小，苦乐不均；第五件，家人豪纵，有脸者不服钤束，无脸者不能上进"（013-023） |
| 3 | 获取[[对牌]] | 获取最高授权凭证，确保命令不可违抗 | "[[贾珍]]便忙向袖中取了[[宁国府]][[对牌]]出来"（013-021） |
| 4 | 心理战—制造敬畏 | 利用来人传递威名，使下属提前产生敬畏 | [[宁国府]]都[[总管]][[赖升|来升]]传齐众人道："那是个有名的烈货，脸酸心硬，一时恼了，不认人的"（014-001） |
| 5 | 公开宣示 | 向全体宣布新规则和执行态度，不留余地 | "既托了我，我就说不得要讨你们嫌了……如今可要依著我行，错我半点儿，管不得谁是有脸的，谁是没脸的，一例现清白处理"（014-002） |
| 6 | 花名册查验 | 按名逐一排查人员，了解队伍构成 | "吩咐[[彩明]]念花名册，按名一个一个的唤进来看视"（014-002） |
| 7 | 明确分工 | 将人员分组定岗，职责具体到人、到物 | "这二十个分作两班，一班十个，每日在里头单管人客来往倒茶……这四个人单在内[[茶房]]收管杯碟茶器，若少一件，便叫他四个描赔"（014-003） |
| 8 | 登记造册 | 一切物资出入登记在册，堵住私拿冒领 | "一面交发，一面提笔登记，某人管某处，某人领某物，开得十分清楚"（014-004） |
| 9 | 定时点卯 | 设定严格的时间表和打卡制度 | "卯正二刻我来点卯，巳正吃早饭，凡有[[领牌]][[回事]]的，只在午初刻。戌初烧过黄昏纸，我亲到各处查一遍"（014-003） |
| 10 | 杀一儆百 | 严惩第一个违规者，以儆效尤 | "登时放下脸来，喝命：'带出去，打二十[[板子]]！'……'出去说与[[赖升|来升]]，革他一[[月钱制度|月银]]米！'……'明日再有误的，打四十，后日的六十，有要挨打的，只管误！'（014-011） |
| 11 | 以身作则 | 管理者亲自勤勉，不偷安推托 | "[[王熙凤|凤姐]]不畏勤劳，天天于卯正二刻就过来点卯理事"（014-005）；"[[王熙凤|凤姐]]见如此，心中倒十分欢喜，并不偷安推托，恐落人褒贬"（014-018） |

## 核心原则

| 原则 | 说明 | 原文例证 |
|------|------|----------|
| 全盘掌控 | 不放过任何一个细节，从人到物全部过手 | "[[彩明]]钉造簿册……要家口花名册来查看"（014-002） |
| 权责对等 | 给职责同时给工具，出问题追责到人 | "某人守某处，某处所有[[桌椅]][[古董]]起……或丢或坏，就和守这处的人算帐描赔"（014-003） |
| 制度化 | 用制度管人，而不是用人管人 | "如今都有定规，以后那一行乱了，只和那一行说话"（014-003） |
| 循序渐进 | 从诊断到执行有条不紊，不急不躁 | 先理头绪→诊断→分派→执行→纠偏（013-022）至（014-011） |
| 公私分明 | 严格执法但不影响[[人情]]往来 | 既打了人又给[[贾珍]]送粥（014-005）；执法同时兼顾荣宁两府事务（014-018） |

## 核心原文

> 这里凤姐儿来至三间一所抱厦内坐了，因想：头一件是人口混杂，遗失东西，第二件，事无专执，临期推委，第三件，需用过费，滥支冒领，第四件，任无大小，苦乐不均，第五件，家人豪纵，有脸者不服钤束，无脸者不能上进。此五件实是宁国府中风俗，不知凤姐如何处治，且听下回分解。（013-023）

> 凤姐即命彩明钉造簿册。即时传来升媳妇，兼要家口花名册来查看，又限于明日一早传齐家人媳妇进来听差等语。（014-002）

> 既托了我，我就说不得要讨你们嫌了。我可比不得你们奶奶好性儿，由著你们去。再不要说你们『这府里原是这样』的话，如今可要依著我行，错我半点儿，管不得谁是有脸的，谁是没脸的，一例现清白处理。（014-002）

> 登时放下脸来，喝命：「带出去，打二十板子！」一面又掷下宁国府对牌：「出去说与来升，革他一月银米！」……凤姐道：「明日再有误的，打四十，后日的六十，有要挨打的，只管误！」（014-011）
