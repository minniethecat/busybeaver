# 人物父母子女

《[[红楼梦]]》人物的亲子关系是理解家族制度和代际传承的基础。以下分父系和母系两个子表展示。

## 父系（人物与其父亲）

以下人物在页面上记录了父亲信息，按父亲姓名聚合排列，可直观看到同父子女群组：

::: query
father: NOT NULL
display: table
fields: [label, father, mother, tags]
sort: father
order: asc
field_labels:
  label: 子女
  father: 父亲
  mother: 母亲
  tags: 身份
:::

## 母系（人物与其母亲）

以下人物在页面上记录了母亲信息，按母亲姓名聚合排列：

::: query
mother: NOT NULL
display: table
fields: [label, mother, father, tags]
sort: mother
order: asc
field_labels:
  label: 子女
  mother: 母亲
  father: 父亲
  tags: 身份
:::

## 说明

- 同一个人物可能同时出现在父系和母系表中。
- 如有父无母（或有母无父），表示对应页面尚未补充完整信息。
- 数据来源为各人物页面的 frontmatter `father` / `mother` 字段。
