# 芦雪广联句

《[[芦雪广联句]]》是[[大观园诗社]]在[[芦雪广]]举行的一场联句活动（第050回），参与人数最多、竞争最激烈。[[王熙凤|凤姐]]起句"一夜北风紧"，[[史湘云]]最为活跃，"扬眉挺身"抢联，成为公认的联句冠军。

## 特点

此联句参与者众多，竞争激烈。[[史湘云]]最为活跃，"扬眉挺身"抢联（050回），成为公认的联句冠军。"一夜北风紧"一句，[[王熙凤|凤姐]]自谦"粗鄙"，众人却赞"这正是会作诗的起法"（050回）。

联句后众人又作[[咏红梅花]]，[[贾宝玉|宝玉]]联句落第被罚往[[栊翠庵]]向[[妙玉]]乞[[红梅]]。

**注析**（择要）
- **一夜北风紧**：[[王熙凤|凤姐]]起句，粗朴直白却有气势，为众人联句留出广阔空间，[[脂批]]赞"这正是会作诗的起法"。
- **匝地惜琼瑶**：琼瑶为美玉，喻白雪之洁白珍贵。"匝地"写雪落满地，"惜"字含怜惜之意。
- **葭动灰飞管**：用古代律管候气之典。[[冬至]]时律管中葭灰飞动，表示阳气始生，[[节气]]转换。
- **阳回斗转杓**：北斗七星之杓转向，阳气回生。与上句皆写季节变迁，由冬入春。
- **鳌愁坤轴陷**：巨鳌担忧大地塌陷。用《列子·汤问》巨鳌背负大地的神话，喻雪厚似压大地。
- **吟鞭指灞桥**：用唐代诗人郑綮"诗思在灞桥风雪中驴背上"之典，写雪中诗兴。
- **赐裘怜抚戍**：赏赐裘衣以抚慰戍边将士，暗含对边关将士的体恤之情。
- **加絮念征徭**：给征夫棉衣多加棉絮。用唐代宫女为征人缝衣的故事，写雪天念及征人之寒。
- **煮芋成新赏**：用唐代懒残禅师煨芋之典，或指雪天煮芋为冬日[[游戏娱乐|雅趣]]。
- **撒盐是旧谣**：用《世说新语》谢朗"撒盐空中差可拟"喻雪之典，呼应前代咏雪名句。
- **伏象千峰凸**：雪覆群峰，如千头白象伏卧，比喻雄奇生动。
- **盘蛇一径遥**：雪中小路蜿蜒如蛇行，意象逼真。
- **僵卧谁相问**：用[[东汉]]袁安大雪天僵卧家中、不愿扰人的[[典故]]，写雪中高士之清节。
- **狂游客喜招**：用[[东晋]]王徽之雪夜乘舟访戴逵、至门不入的[[典故]]，写雪中雅兴。
- **天机断缟带**：天机织出的白绢断裂，喻大雪漫天飞舞，想象奇丽。
- **海市失[[鲛绡]]**：海市蜃楼中鲛人所织的绡纱失落，亦喻白雪之无边无际。
- **[[月窟]]翻银浪**：月宫中翻涌银色波浪，喻雪光月色交相辉映。
- **霞城隐赤标**：赤城山被雪覆盖而隐去红色的峰顶，写雪之覆盖力。
- **沁梅香可嚼**：雪水沁入[[梅花]]，香气仿佛可咀嚼。意境极美，为全联句名句之一。
- **凭诗祝舜尧**：以诗祝愿天下太平如[[尧舜]]之世，收束全篇，寓颂于雅。

## 叙事功能

"芦雪广联句"在《[[红楼梦]]》叙事体系中承担着"联句作为诗社叙事的竞争性高潮：从合作到竞赛的叙事节奏转换及其人物性格显影"功能。《[[红楼梦]]》中的诗社活动从第三十七回海棠社开始到第七十回柳絮词结束，共经历约六次大型诗会：白海棠诗（第三十七回，含蓄、内敛，众人初次见面、彼此试探）、菊花诗（第三十八回，高雅、深沉，以菊喻人、各写其心）、螃蟹咏（第三十八回，谐谑、讽刺，宝钗的"食螃蟹绝唱"）、芦雪广联句（第五十回，激烈、竞技，众人争相吟咏、互不相让）、柳絮词（第七十回，感伤、告别，最后一次诗社活动）。在这六次诗会中，芦雪广联句是唯一以"联句"形式进行的。其他诗会都是"每人各作一首"（各自独立创作、互不干扰），芦雪广则是"众人合写一首"（每人一句、接续而上）。叙事者在第五十回选择了联句这一形式，不只在写诗的内容（写雪景），更在写诗的过程（写众人如何联句）。

"联句"本身就是叙事的内容：[[史湘云]]"扬眉挺身"抢联（050回），把自己的诗一句接一句地往联句里塞。湘云的"抢"是叙事者展示她的"豪爽"和"好胜"的最直接的方式（不需要叙事者说"湘云好胜"，读者看到她在联句中"抢"的样子就知道了）。

[[林黛玉]]也不甘示弱（黛玉平时在诗会上虽表现突出但从不"抢"。菊花诗时她是"等着"别人先写的、她是"逼出来"的，但她在联句中也在"抢"，因为联句是你一句我一句、不"抢"就没有机会了。这暴露了黛玉性格中被压抑的"好胜心"，诗是她唯一不让人的地方）。[[贾宝玉]]则"落第"（他联句最少、质量最差，被罚去[[栊翠庵]]向[[妙玉]]乞[[红梅]]）。叙事者通过联句的形式（"谁抢了多少句、谁说了什么"），在叙事中完成了对人物性格的"竞争性显影"：联句中"抢"得最多的是湘云（直率、好胜、精力充沛），"不抢"但"句句精彩"的是黛玉（内敛但自信，不争数量争质量），"抢不到"的是宝玉（缺乏诗才）。联句的形式本身即为叙事的内容。叙事者不需要说"湘云好胜"，只需要写她"扬眉挺身"。这是"表演性叙事"（联句本身就是表演）的极致。

"芦雪广联句"的第二叙事功能在于"凤姐起句'一夜北风紧'在叙事中的'粗人雅事'叙事意义：不识字者的文学参与作为全书'文野'主题的最后一次和解"。《[[红楼梦]]》中[[王熙凤]]是全书最核心的"不识字"人物之一。她是荣国府的当家奶奶但不通文墨（第二十八回凤姐说"我又不会作什么湿的干的"，028-009。她在诗社活动中从不主动作诗、也从不对诗作发表意见）。但在第五十回芦雪广联句中，凤姐主动起了第一句："一夜北风紧"。这四个字朴素直白、无修辞无典故，但被众人赞为"这正是会作诗的起法"（050回）。

凤姐的起句在叙事上承担了多重功能。第一，凤姐的"一夜北风紧"是全书唯一一次凤姐的"文学参与"（也是全书唯一一个不识字的粗人参与了诗的创作）。

第二，凤姐的起句"好"不是因为文采好（凤姐有没有文采都是问题），而是因为"起得好"。"一夜北风紧"给后续联句留下了最大的空间（如果凤姐起句写得太具体，比如"雪压南山松"，后面的人就很难接；但凤姐起句"一夜北风紧"只写了"起风了"，后面的人可以写雪、可以写寒、可以写暖、可以写任何与"北风"相关的内容）。

第三，凤姐的起句在叙事上实现了"文学雅事"与"世俗日常"的一次"和解"。凤姐代表了"世俗"（功利的、算计的、不懂文学的），诗代表了"雅"（无功利的、审美的、文学的），两者在芦雪广联句中通过"一夜北风紧"得到了和解（凤姐用最朴素的四个字参与了最高雅的诗会，"粗人"和"雅事"在叙事中互不排斥）。叙事者安排"粗人起句"（凤姐，不识字的俗人）和"诗人收束"（李绮，凭诗祝舜尧，以最雅的祝词收尾），在叙事上形成了一个由"俗"入"雅"的叙事弧线。凤姐的"北风紧"把读者带入了一个风雪交加的世界，众人的联句层层展开、越来越雅、越来越深、直到"凭诗祝舜尧"的太平颂歌。读者在联句中经历了一次从"听凤姐说天气"到"进入诗的境界"的叙事过山车。
