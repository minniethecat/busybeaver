# 红楼夫妻

《[[红楼梦]]》中人物的婚姻关系是理解家族联姻、权力格局和[[人物结局|人物命运]]的重要线索。以下列出所有有配偶记载的人物及其婚姻关系。

由于数据双向记录，夫妻双方各自出现，可通过表格两侧交叉对照。

## 配偶一览

::: query
spouse: NOT NULL
display: table
fields: [label, gender, spouse, tags]
sort: label
order: asc
field_labels:
  label: 人物
  gender: 性别
  spouse: 配偶
  tags: 身份
:::

## 说明

1. 妾室（如[[平儿]]、[[袭人]]、[[香菱]]等）也收录于 spouse 字段中，反映小说时代的多妻制婚姻形态。
2. 部分人物的 spouse 数据为单向记录：若某人的配偶在表中未见反向条目，说明该配偶页面尚未补充关系数据，欢迎补充。
3. 本表数据来源为各人物页面的 frontmatter `spouse` 字段，由注册表自动维护。
