# 凤姐借刀杀人法

[[王熙凤]]得知[[贾琏]]偷娶[[尤二姐]]后，先用"贤良"姿态将[[尤二姐]]接入府中，再借[[秋桐]]之手日夜折磨[[尤二姐]]，自己抽身事外、[[坐山观虎斗]]。此法体现了借他人之力除掉对手而不脏自己的手的权谋之术。

## 技能定义

| 字段 | 值 |

| 名称 | 凤姐借刀杀人法 |
| 出处 | [[第069回]] |
| 创始者 | [[王熙凤]] |
| 类型 | 权谋·借力除敌 |
| 核心原则 | 引第三方为刀、己不沾手、坐观成败 |
| 适用级别 | 需要除掉对手但不宜亲自出手时 |
| 复杂度 | 高（需对局势有精确的判断和控制） |

## 适用条件

| 条件 | 说明 |
|------|------|
| 适用场景 | 目标与自己有利益冲突但不能直接动手时；有可借用的第三方"刀"时 |
| 前提条件 | 有一把"刀"（即可被利用的第三方）且该刀与目标已有或可有矛盾 |
| 风险因素 | 刀可能反噬；计划可能被识破；涉及人命时后果不可控 |
| 所需资源 | 对局势的精确判断、对"刀"的心理操纵能力、置身事外的演技 |
| 道德评价 | 此计在原著中以悲剧告终，属典型的阴谋之术，道德风险极高 |

## 核心原则

| 原则 | 说明 | 原文例证 |
|------|------|----------|
| 不沾手 | 从始至终自己不出面做恶事，全部借他人之手完成 | [[王熙凤|凤姐]]装病不出，全是[[秋桐]]在骂、[[婆子]]在踩（069-005）（069-006） |
| 占据道德高地 | 表面上做足善人姿态，让外人无可指责 | [[王熙凤|凤姐]]在[[贾母]]面前贤良无比，[[贾母]]称赞"[[王熙凤|凤丫头]]倒好意待他"（069-006） |
| 利用矛盾 | 找到"刀"与目标之间天然的矛盾点（争宠、嫉妒等），善加利用 | [[秋桐]]年轻气盛、嫉妒[[尤二姐]]得宠，天然就是好刀（069-006） |
| 留有后手 | 随时准备清除"刀"，防止刀口转向自己 | [[王熙凤|凤姐]]计划在[[尤二姐]]死后除去[[秋桐]]（069-006） |

## 核心原文

> 凤姐虽恨秋桐，且喜借他先可发脱二姐，自己且抽头，用"借剑杀人"之法，"坐山观虎斗"，等秋桐杀了尤二姐，自己再杀秋桐。主意已定。（069-006）

> 凤姐既装病，便不和尤二姐吃饭了。每日只命人端了菜饭到他房中去吃，那茶饭都系不堪之物。平儿看不过，自拿了钱出来弄菜与他吃……（069-005）

> 没人处常又私劝秋桐说："你年轻不知事。他现是二房奶奶，你爷心坎儿上的人，我还让他三分，你去硬碰他，岂不是自寻其死？"那秋桐听了这话，越发恼了，天天大口乱骂。（069-006）

> 凤姐一面使人暗暗调唆张华，只叫他要原妻……又调唆张华："亲原是你家定的，你只要亲事，官必还断给你。"（069-002）
