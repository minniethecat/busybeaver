# 人物结局

《[[红楼梦]]》中人物的最终归宿是小说的核心主题之一，"[[树倒猢狲散]]"、"[[白茫茫大地真干净]]"。以下列出有明确结局记载的人物及其命运。

## 人物结局一览

::: query
fate: NOT NULL
display: table
fields: [label, fate, tags]
sort: label
order: asc
field_labels:
  label: 人物
  fate: 结局
  tags: 身份
:::

## 说明

1. 结局描述以[[程甲本]]一百二十回为主要依据，辅以[[脂批]]线索。部分人物在[[程甲本]]与[[脂批]]中的结局有差异（如[[史湘云]]、[[贾巧姐]]等）。
2. 数据来源为各人物页面的 frontmatter `fate` 字段，由注册表自动维护。尚未补充结局的人物不会出现在上表中。
3. 本表仅收录有明确结局描述的人物，部分次要人物原著未交代结局，故不在此列。
