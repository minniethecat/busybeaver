# 曹寅

曹寅（1658–1712），字子清，号楝亭，[[曹雪芹]]之祖父。历任内务府[[郎中]]兼[[江宁织造]]、通政使司通政使等职，是康熙[[皇帝]]信重的包衣重臣。

## 生平大事

| 时期 | 大事 |
|------|------|
| 1658年 | 出生，隶属汉军正白旗包衣 |
| 1690-1712年 | 任[[江宁织造]]，为康熙[[皇帝]]信重的江南耳目 |
| 康熙年间 | 主持刊刻《[[全唐诗]]》与《佩文韵府》等大型书籍 |
| 1712年 | 逝世于[[维扬|扬州]]，身后曹家渐趋衰落 |

曹寅主持刊刻《[[全唐诗]]》《佩文韵府》等大型书籍，家中藏书极富。其家世背景（[[江宁织造|江宁织造府]]的盛衰）和丰富的藏书、诗文创作，为[[曹雪芹]]创作《[[红楼梦]]》提供了深厚的家族记忆与文学土壤。

## 历史评价

曹寅是曹家由盛转衰的关键人物。他生前达到了曹氏家族权势的顶峰：兼任江宁织造、两淮巡盐御史，是康熙帝在江南最信任的耳目。康熙六次南巡，曹寅接驾四次，这正是《红楼梦》中赵嬷嬷所说的"咱们贾府正在姑苏扬州一带监造海舫，修理海塘，只预备接驾一次"（016-011）的现实底本。然而接驾的巨大开销也为曹家埋下了财务危机的伏笔。曹寅死后，曹家亏空日积月累，最终在雍正年间被抄家。可以说，《红楼梦》中贾府由盛而衰的叙事框架，其基本的情感结构来自曹雪芹对祖父时代由繁华到败落的家族记忆。曹寅主持刊刻《全唐诗》的文化事业，也是曹雪芹"诗礼簪缨"之族文化底蕴的历史源头。
