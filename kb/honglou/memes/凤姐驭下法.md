# 凤姐驭下法

[[凤姐驭下法]]出自第十三至十四回"[[王熙凤协理宁国府]]"，是[[王熙凤]]在代理[[宁国府]]丧事管理中展现的系统化管理方法。此法以"立规矩。明责任。严惩戒。恩威并施"为核心，展示了在混乱组织中重建秩序的铁腕管理术。

## 技能定义

| 字段 | 值 |
|------|-----|
| 名称 | 凤姐驭下法 |
| 出处 | [[第013回]]至[[第014回]]（[[王熙凤协理宁国府]]） |
| 创始者 | [[王熙凤]] |
| 类型 | 管理·权术·执行力 |
| 核心原则 | 先定规矩、划清责任、严惩不贷、恩威并行 |
| 适用对象 | 接手混乱组织/项目，需快速建立秩序的管理者 |
| 复杂度 | 中 |

## 适用条件

| 条件 | 说明 |
|------|------|
| 适用场景 | 接管一个管理松懈、责任不清的组织或项目 |
| 前提条件 | 获得充分的授权（来自上级的明确背书） |
| 风险因素 | 执行过严引发集体不满、触及既得利益群体反弹 |
| 所需资源 | [[对牌]]等执行权力、准确的时间管理系统、得力的助手 |
| 成功标志 | 混乱状态明显改观，人人各司其职，令行禁止 |

## 操作步骤

| 步骤 | 名称 | 操作说明 | 原文体现 |
|------|------|----------|----------|
| 1 | 摸底诊断 | 到任前先分析问题根源，找出五大弊病 | [[王熙凤|凤姐]]到[[宁国府]]前已看清五大弊端：人口混杂遗失东西、事无专管推诿、需用过费滥支冒领、任无大小苦乐不均、家人豪纵不服约束（013-023） |
| 2 | 立规明责 | 宣布新规矩，明确每个人的责任范围和考核标准 | [[王熙凤|凤姐]]到任后先"按数造册"，分派任务："这二十个分作两班"、"这二十个每日轮流"（014-003），人人有专责 |
| 3 | 严惩违纪 | 第一个违规者从重处罚，以此立威 | 一人迟到，[[王熙凤|凤姐]]"放下脸来，喝命：'带出去，打二十[[板子]]！'"又"革他一[[月钱制度|月银]]米"（014-011），众人"又见[[王熙凤|凤姐]]眉立，知是恼了" |
| 4 | 公正行赏 | 对办事得力者给予适当肯定，保持组织正向激励 | [[王熙凤|凤姐]]对[[张材家的]]等按程序发放工银（014-010），对[[来旺家的|来旺媳妇]]等依例支给，一切照章办事 |

## 核心原则

| 原则 | 说明 | 原文例证 |
|------|------|----------|
| 先诊后治 | 不动则已，动手前必先摸清病灶 | [[王熙凤|凤姐]]在接手前已将[[宁国府]]五件弊病"一一搜求"清楚（013-023），对症下药 |
| 规矩先行 | 规矩摆在[[人情]]之前，一切按新规执行 | "素日跟我的人，随身自有[[钟表]]，不论大小事，我是皆有一定的时辰"（014-005），以制度代替人治 |
| 杀一儆百 | "头一次宽了，下次人就难管"。第一次违规必须重罚 | "登时放下脸来，喝命：'带出去，打二十[[板子]]！'"（014-011），从此众人凛然 |
| 恩威并行 | 严而不苛，威而不暴，罚后给希望 | [[王熙凤|凤姐]]处置完违纪者后仍照常派发各人工银（014-010），不因一人之过而迁怒众人 |

## 核心原文

> 这里[[凤姐]]儿来至[[宁国府]]口三间一所抱厦内坐了，因想：头一件是人口混杂，遗失东西，第二件，事无专管，临期推诿，第三件，需用过费，滥支冒领，第四件，任无大小，苦乐不均，第五件，家人豪纵，有脸者不服钤束，无脸者不能上进。此五件实是[[宁国府]]中风俗。 （013-023）

> 这三十个每日轮流各处上夜，照管门户，监察火烛，打扫地方。这下剩的按著房屋分开，某人守某处，某处所有桌椅古董起，至于痰盒掸帚，一草一苗，或丢或坏，就和守这处的人算账描赔。 （014-003）

> [[凤姐]]便说道：「明儿他也睡迷了，后儿我也睡迷了，将来都没了人了。本来要饶你，只是我头一次宽了，下次人就难管，不如现开发的好。」登时放下脸来，喝命：「带出去，打二十板子！」一面又掷下[[宁国府]][[对牌]]：「出去说与来升，革他一月银米！」众人听说，又见[[凤姐]]眉立，知是恼了，不敢怠慢。 （014-011）

> [[凤姐]]缓缓向[[宝玉]]问道：「[[宝兄弟]]，你侄媳妇少你什么？」[[宝玉]]满脸陪笑说：「他也不知钱，我也不用他的。」（014-013）

## 应用参考

| 场景 | 建议调整 |
|------|----------|
| 现代团队管理 | 摸底诊断步骤通过数据分析/访谈完成；立规矩对应制定明确的OKR和考核制度 |
| 接手烂摊子项目 | 杀一儆百法仍然适用：找一个最明显的违规案例从严处理，快速建立威信 |
| 创业公司制度化 | 第一步摸底最重要，小公司不宜全盘照搬，选最关键的一两个问题先解决 |

## 补充说明

补充说明：凤姐驭下法的核心并非严苛，而是"公平"二字。她定规矩人人一样，罚违规不[[徇私]]情，支工银照章办事。正因为公正，严苛才被接受。
