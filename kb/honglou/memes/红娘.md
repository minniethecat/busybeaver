# 红娘

红娘是《[[西厢记]]》中[[崔莺|崔莺莺]]的婢女，为[[张生]]与[[会真记|莺莺传]]书递简。书中三处提及。

## 生平大事

| 时期 | 大事 |
|------|------|
| 唐代（故事背景） | [[崔莺莺]]婢女，为[[张生]]与[[会真记|莺莺传]]书递简，事见《[[西厢记]]》 |
| 书中第001回 | [[石头]]自述此书写法不俗，不套"红娘[[霍小玉|小玉]]"之旧稿（001-013） |
| 书中第005回 | [[秦可卿]]房中[[陈设]]有"红娘抱过的鸳枕"（005-010） |
| 书中第040回 | [[黛玉]][[牙牌令]]引《[[西厢记]]》曲文"[[纱窗]]也没有红娘报"（040-016） |
| 书中第058回 | [[麝月]]以"[[拷打]]红娘"打趣[[芳官]]（058-012） |

第一回[[石头]]自述此书写法不俗：

> 不比那些胡牵乱扯，忽离忽遇，满纸才人淑女，子建文君红娘小玉等通共熟套之旧稿。
> （001-013）

第五回[[秦可卿]]房中以红娘抱过的鸳枕为[[陈设]]：

> 亲自展开了西子浣过的纱衾，移了红娘抱过的鸳枕。
> （005-010）

第四十回[[林黛玉|黛玉]][[牙牌令]]引《[[西厢记]]》曲文：

> 纱窗也没有红娘报。
> （040-016）

第五十八回[[芳官]]因丧期不施[[脂粉]]，被[[麝月]]取笑：

> 把一个莺莺小姐，反弄成拷打红娘了！
> （058-012）

"红娘"在《[[红楼梦]]》中既是《[[西厢记]]》[[典故]]的化身，也被[[曹雪芹]]用以构建小说间的互文，从[[石头]]自序的翻案，到[[秦可卿]]房中的铺陈，再到[[林黛玉|黛玉]][[麝月]]口中的日常引用，贯穿全书始终。

## 在书中的体现

红娘在《[[红楼梦]]》中被提及四次，分布在全书的不同回目和叙事层面：第一回[[石头]]自述本书写法不俗，"不套红娘[[霍小玉|小玉]]等通共熟套之旧稿"（001-013），第五回[[秦可卿]]房中[[陈设]]"红娘抱过的鸳枕"（005-010），第四十回[[林黛玉|黛玉]][[牙牌令]]引[[西厢记]]曲文"[[纱窗]]也没有红娘报"（040-016），第五十八回[[麝月]]以"把一个莺莺小姐，反弄成[[拷打]]红娘了"打趣[[芳官]]（058-012）。四次提及从作者自述、环境铺陈、人物语言到日常玩笑，红娘这一文学[[典故]]以四种不同的方式融入《[[红楼梦]]》的叙事肌理，成为全书最重要的小说互文符号之一。
