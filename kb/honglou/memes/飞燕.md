# 飞燕

汉成帝皇后赵飞燕，以体轻善舞闻名。书中两处提及。

## 在书中的体现

飞燕在《[[红楼梦]]》中见于第五回和第二十三回。第五回[[秦可卿]]房中[[陈设]]中有"飞燕立著舞过的金盘"（005-010），以飞燕之舞姿轻盈烘托[[秦可卿]]房内香艳旖旎的氛围。第二十三回[[贾宝玉|宝玉]]得见飞燕等外传小说（023-029），飞燕在此作为古代美人[[典故]]出现，与[[合德]]、[[武则天]]、[[杨贵妃]]等并列，构成小说中"红颜"意象群的一部分。

## 生平大事

| 时期 | 大事 |
|------|------|
| 西汉成帝时 | 入宫为婕妤，后立为皇后，以体轻善舞闻名 |
| 西汉哀帝时 | 被废为庶人，[[自杀]] |
| 书中第005回 | [[秦可卿]]房中[[陈设]]中有"飞燕立著舞过的金盘"（005-010） |
| 书中第023回 | [[宝玉]]得见飞燕等外传小说（023-029） |

第五回[[秦可卿]]房中[[陈设]]：

> 一边摆著飞燕立著舞过的金盘。
> （005-010）

第二十三回[[贾宝玉|宝玉]]得见飞燕等外传小说：

> 便把那些古今小说并那飞燕，合德，武则天，杨贵妃的外传与那传奇角本买了许多来，引宝玉看。
> （023-029）

飞燕之妹[[合德]]亦同列其中，[[兄弟姐妹关系|姊妹]]二人皆以美色得宠于汉成帝，后世常并称"飞燕[[合德]]"以喻妖娆宠妃。
