# 兄弟姐妹关系

《[[红楼梦]]》中的兄弟姐妹关系是人物网络的重要组成部分，既有同父同母的嫡出手足，也有庶出、同父异母等复杂情况。

以下列出所有在页面上记录了兄弟姐妹的人物。

## 手足关系一览

::: query
siblings: NOT NULL
display: table
fields: [label, siblings, father, mother, tags]
sort: label
order: asc
field_labels:
  label: 人物
  siblings: 兄弟姐妹
  father: 父亲
  mother: 母亲
  tags: 身份
:::

## 说明

1. **同父异母**：结合「父亲」和「母亲」两列可判断，若两列不同，须注意嫡庶之别。例如[[贾探春]]与[[贾宝玉]]同父（[[贾政]]）而异母（[[赵姨娘]] vs [[王夫人]]）。
2. **双向记录**：手足关系通常双向记录（A 的 siblings 含 B，B 的 siblings 含 A），但部分人物可能仅单向记录，欢迎补充。
3. **堂表亲不在本表**：[[贾宝玉]]与[[林黛玉]]是表亲，与[[贾珍]]是堂亲，这些关系记录在 `cousins` 字段，不在本表 siblings 中。
4. 数据来源为各人物页面的 frontmatter `siblings` 字段。
