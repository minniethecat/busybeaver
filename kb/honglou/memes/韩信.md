# 韩信

西汉开国名将，[[淮阴]]人。第六十八回[[凤姐大闹宁国府]]时，以韩信自喻。

## 生平大事

| 时期 | 大事 |
|------|------|
| 秦末 | 从[[项羽]]不受重用，后投[[刘邦]]，被拜为大将 |
| 楚汉之际 | 暗度陈仓、背水一战、四面楚歌，助[[刘邦]]灭[[项羽]]，封[[淮阴]]侯 |
| 西汉初年 | 被吕后与[[萧何]]设计诱杀 |
| 书中第068回 | [[王熙凤]]大闹[[宁国府]]时以韩信自喻智谋（068-014） |

> 嫂子说，我便是个韩信张良，听了这话，也把智谋吓回去了。
> （068-014）


## 在书中的体现

韩信在《[[红楼梦]]》第六十四回中出现在[[林黛玉|黛玉]]的《[[五美吟]]》之三。"黥布韩信一般才"（064-005）。

[[林黛玉|黛玉]]以"长揖雄谈态自殊，美人巨眼识穷途"咏[[红拂]]，以"尸居余气杨公幕，岂得羁縻女丈夫"赞[[红拂]]的识人之明。《[[五美吟]]》中提到的历史人物包括[[西施]]、[[虞姬]]、[[王嫱|明妃]]（[[王昭君]]）、[[绿珠]]、[[红拂]]。韩信作为[[红拂]]诗中的[[典故]]人物（"黥布韩信一般才"），与黥布并列为汉初功臣，喻[[红拂]]识人的眼光：她看出[[李靖]]不是池中之物，就像[[刘邦]]的谋士看出韩信、黥布是人才一样。

韩信在书中是"怀才不遇"与"功高被戮"的双重符号。与[[林黛玉|黛玉]]《[[五美吟]]》中五位薄命红颜的"才高命蹇"形成了跨越性别的共鸣。

[[林黛玉|黛玉]]以韩信入诗，是她"[[题帕三绝]]"（034-005）之外又一次借古人抒己怀。"一般才"三字既是咏史也是自况。
