# 地点分类大全

《[[红楼梦]]》中的地点涵盖[[大观园]]的亭台楼阁、[[荣国府]]与[[宁国府]]的院落厅堂，以及[[金陵]]、[[姑苏]]等外部世界。

以下按地点类型分类展示，每类独立查询块。

## 园林

::: query
type: place
place_type: 园林
display: table
fields: [label, region, tags]
sort: label
title: 园林
field_labels:
  label: 地点
  region: 所属
  tags: 标签
:::

## 说明

- 地点类型由 `place_type` 字段自动分类：园林、建筑、居住、宗教、自然、行政区划、商业。
- "所属" 列显示该地点的上级归属（如[[潇湘馆]]。 [[大观园]]）。
- 有今地名考据的地点会在详情页面中标注。
