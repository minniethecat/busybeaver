# 高鹗

高鹗（约1738—约1815年），字兰墅，清代文学家，是《[[红楼梦]]》[[后四十回]]的主要续补者和整理者。

## 生平大事

| 时期 | 大事 |
|------|------|
| 约1738年 | 出生，字兰墅 |
| 乾隆年间 | 在京师与[[程伟元]]相识 |
| 1791年（乾隆五十六年） | 与[[程伟元]]合作整理刊印一百二十回《[[红楼梦]]》（[[程甲本]]） |
| 约1815年 | 逝世 |

## 续补红楼梦

高鹗与[[程伟元]]合作，在前八十回基础上整理续补[[后四十回]]，于乾隆五十六年（1791年）刊印[[程甲本]]。[[后四十回]]是否全由高鹗独立创作，学界存有争议，有观点认为其中保留了[[曹雪芹]]部分残稿。

## 历史评价

高鹗在红学史上是一个争议人物。肯定者认为，没有高鹗的续补，一百二十回全本《红楼梦》不可能在乾隆年间就以印本形式广泛流传，《红楼梦》的传播史将完全是另一番面貌。批评者则认为，后四十回无论在人物性格（如宝玉热衷科举）、情节走向（如"兰桂齐芳"的团圆结局）还是艺术水准上，都与前八十回存在显著差距，高鹗"续貂"之作扭曲了曹雪芹的原意。胡适以来的现代红学对后四十回的评价尤为严厉。但不可否认的是，正是程高本的出版使《红楼梦》超越了抄本时代，成为一部全国性的文学作品。这一历史功绩使得关于高鹗的讨论，始终在"功臣"与"罪人"的两个极端之间摇摆。
