# 探春承包法

[[贾探春]]代理[[荣国府]]家政期间，受[[赖大家]]花园承包经验的启发，在[[大观园]]推行承包责任制改革。此法以"使之以权、动之以利"为核心，将园中资源分包给老妈妈们经营，兼顾效率与公平，实现了多赢。

## 技能定义

| 字段 | 值 |
|------|-----|
| 名称 | 探春承包法 |
| 出处 | [[第056回]] |
| 创始者 | [[贾探春]]、[[薛宝钗]]、[[李纨]]、[[平儿]]（四人议定） |
| 类型 | 管理·激励设计·制度创新 |
| 核心原则 | 使之以权、动之以利；兼顾效率与公平 |
| 适用级别 | 组织内部资源盘活、激励机制设计 |
| 复杂度 | 较高 |

## 适用条件

| 条件 | 说明 |
|------|------|
| 适用场景 | 组织内部有闲置资源、浪费严重、成本虚高、员工动力不足时 |
| 前提条件 | 对现有问题有充分调研；有外部成功案例可参考 |
| 风险因素 | 触动既得利益者可能引发反弹；承包者见利忘义可能产生新问题 |
| 所需资源 | 明确的资源清单、合格的承包人选、公平的分配方案 |
| 成功标志 | 效率提升、成本降低、各方满意 |

## 操作步骤

| 步骤 | 名称 | 操作说明 | 原文体现 |
|------|------|----------|----------|
| 1 | 发现问题 | 从账目细节中发现重复支出、资源浪费 | "因此我心中不自在。钱费两起，东西又白丢一半，通算起来，反费了两折子"（056-002） |
| 2 | 借鉴经验 | 参考外部成功案例，转化应用 | "年里往[[赖大家]]去……谁知那么个园子……一年还有人包了去，年终足有二百两银子剩。从那日我才知道，一个破荷叶，一根枯草根子，都是值钱的。"（056-002） |
| 3 | 确立原则 | 明确改革的多重目标 | "一则园子有专定之人修理，[[花木]]自有一年好似一年的……二则也不至作践，白辜负了东西，三则老妈妈们也可借此小补……四则亦可以省了这些花儿匠山子匠打扫人等的工费。"（056-004） |
| 4 | 选对人 | 选择本分老诚、有专业技能的人承包 | "在园子里所有的老妈妈中，拣出几个本分老诚能知园圃的事，派准他们收拾料理"（056-004）；"这一个老[[祝妈]]是个妥当的，况他老头子和他儿子代代都是管打扫[[竹子]]"（056-006） |
| 5 | 界定权责 | 明确承包范围：定额上缴，超额归己 | "按四季除家中定例用多少外，余者任凭你们采取了去取利，年终算帐"（056-007） |
| 6 | 绕过中间层 | 直接管理，避免中间利益集团盘剥 | "如今这园子里是我的新创，竟别入他们手，每年归帐，竟归到里头来才好"（056-007） |
| 7 | 利益共享 | 承包者有收益的同时，给未承包者分红，减少矛盾 | "他们虽不料理这些，却日夜也是在园中照看当差之人……这园内既有出息，也是分内该沾带些的"（056-007） |
| 8 | 约法三章 | 明确规则和底线，防止乱来 | "不然，我也不该管这事……那时酒醉[[赌博]]生出事来，我怎么见[[赵姨娘|姨娘]]"（056-008） |

## 核心原则

| 原则 | 说明 | 原文例证 |
|------|------|----------|
| 使之以权、动之以利 | 给承包者自主权和经济激励，比行政命令更有效 | "使之以权，动之以利，再无不尽职的了"（056-004，[[李纨]]语） |
| 见利思义 | 在激励的同时设底线，不可一味逐利 | "虽是兴利节用为纲，然亦不可太啬。纵再省上二三百银子，失了大体统也不像"（056-007） |
| 利益均衡 | 让未承包者也能受益，减少改革阻力 | "他们虽不料理这些……这园内既有出息，也是分内该沾带些的"（056-007） |
| [[避嫌]]远疑 | 用间接推荐而非直接安排，避免被人议论 | [[薛宝钗|宝钗]]不肯直接派[[莺儿]]之母，而是推荐[[叶妈]]，让[[叶妈]]找[[莺儿]]妈商量（056-006） |
| 借助权威 | 改革需获得最高层默许和支持 | "虽如此说，也须得回你奶奶一声……若是糊涂多蛊多妒的，我也不肯"（056-004） |

## 核心原文

> 探春道："我因和他家女儿说闲话儿，谁知那么个园子，除他们带的花，吃的笋菜鱼虾之外，一年还有人包了去，年终足有二百两银子剩。从那日我才知道，一个破荷叶，一根枯草根子，都是值钱的。"（056-002）

> 探春因又接说道："不如在园子里所有的老妈妈中，拣出几个本分老诚能知园圃的事，派准他们收拾料理，也不必要他们交租纳税，只问他们一年可以孝敬些什么。一则园子有专定之人修理，花木自有一年好似一年的……二则也不至作践，白辜负了东西，三则老妈妈们也可借此小补，不枉年日在园中辛苦，四则亦可以省了这些花儿匠山子匠打扫人等的工费。"（056-004）

> 李纨笑道："好主意。这果一行，太太必喜欢。省钱事小，第一有人打扫，专司其职，又许他们去卖钱。使之以权，动之以利，再无不尽职的了。"（056-004）

> 宝钗笑道："虽是兴利节用为纲，然亦不可太啬。纵再省上二三百银子，失了大体统也不像……他们虽不料理这些，却日夜也是在园中照看当差之人……这园内既有出息，也是分内该沾带些的。"（056-007）

> 宝钗道："你们只管了自己宽裕，不分与他们些，他们虽不敢明怨，心里却都不服，只用假公济私的多摘你们几个果子，多掐几枝花儿，你们有冤还没处诉。"（056-007）

## 应用参考

| 场景 | 建议调整 |
|------|----------|
| 企业内部改革 | 步骤1-2 = 调研诊断，3 = 顶层设计，4-5 = 试点推行，6-8 = 制度保障 |
| 团队激励 | 可简化为：看清资源→选对人→给自主权→共享收益→定规则 |
| 注意事项 | 必须取得上级充分支持；改革要兼顾各方利益，防止既有利益者破坏；选取的承包人要德才兼备 |
| 与现代管理理论的对应 | "使之以权" = 授权赋能，"动之以利" = 绩效激励，"利益共享" = 利益相关者理论 |
