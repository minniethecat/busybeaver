# 热度排行

## 引用最多 Top 50

按被其他页面引用的次数（total_refs）排序，反映该词条在 Wiki 中的重要程度。

::: query
sort: total_refs
order: desc
limit: 50
display: table
fields: [label, type, total_refs, total_chapters, quality_score]
title: Top 50
:::

## 跨章最广 Top 30

按出现章回数（total_chapters）排序，反映该词条在原著中横跨的篇幅广度。

::: query
sort: total_chapters
order: desc
limit: 30
display: table
fields: [label, type, total_chapters, total_refs, quality_score]
title: Top 30
:::
