# 贾府的门与街道

## 经典路径

### 黛玉初进荣国府（003）
西[[角门]] → [[垂花门]] → [[穿堂]]（[[紫檀]][[插屏]]） → 三间厅 → [[贾母]][[正房]]大院

### 刘姥姥一进荣国府（006）
宁荣街 → 大门石狮子 → [[角门]] → 后街 → [[后门]] → [[倒厅]] → [[影壁]] → [[凤姐院]]

### 元春省亲（018）
西街门 → 大门 → [[仪门]] → 体仁沐德院 → [[大观园]]

### 除夕祭宗祠（053）
[[宁国府|宁府]]大门 → [[仪门]] → [[大厅]] → [[暖阁]] → 内厅 → 内三门 → 内[[仪门]] → 内塞门 → 正堂
