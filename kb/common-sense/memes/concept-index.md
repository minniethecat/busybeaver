# Concept Index

This page aggregates all concept entries in the wiki through a dynamic query block. Each concept below is analyzed in the context of Holbach's materialist critique of religion — from abstract philosophical categories like [[materialism]] and [[determinism]] to specific [[doctrine|theological doctrine]]s like [[trinity]] and [[transubstantiation]]. The table is rendered automatically from page frontmatter.

::: query
type: concept

field_labels:
  label: Concept
  quality: Quality
  total_refs: References
  pn_count: Citations
