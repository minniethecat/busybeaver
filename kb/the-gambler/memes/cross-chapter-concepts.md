# Cross-Chapter Concepts

Some ideas in *The Gambler* appear in a single scene and fade; others thread through the entire novel. This page indexes concepts ranked by how many other pages reference them — a proxy for how widely a theme is cited and therefore discussed across the wiki. The more a concept is referenced, the more it connects different characters, events, and analyses.

The table below updates automatically as concept entries are enriched and cross-referenced.

::: query
type: concept
display: table
fields: [label, total_refs, quality]
field_labels:
  label: Concept
  total_refs: Citations
  quality: Quality Tier
sort: total_refs
order: desc
limit: 30
:::

## About This Index

`total_refs` is a Wiki engine statistic — the number of distinct pages that contain a wikilink to this concept entry. It reflects how often the concept is invoked in discussions of characters, events, and other concepts. A concept with high cross-references (Roulette, Gambling, Zero) is a thematic backbone; a concept with few references is more self-contained.

See [Concept Index](concept-index) for the full alphabetical listing and [Most Cited Pages](most-cited-pages) for the cross-type citation ranking.
