# Index of Events

A catalogue of the key narrative events in *Fifty-One Tales*. Each entry represents a significant story moment — a meeting, a transformation, a prophecy, or a cosmic occurrence — around which a tale turns.

::: query
type: event
display: table
fields: [label, event_type, appears_in_count, quality]
field_labels:
  label: Event
  event_type: Type
  appears_in_count: Appearances
  quality: Quality
sort: label
order: asc
title: All Events
:::

## Types of Events

- **Tale Events** — Central narrative events within individual tales
- **Mythical** — Events referencing classical or invented mythology
- **Cosmic** — Events of universal or eschatological scope
- **Historical** — Events with real-world historical resonance
