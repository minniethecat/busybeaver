# Memory in Yeats

Memory in Yeats is never merely personal reminiscence. It is a conduit to the ancestral past, a portal to the "Great Memory" (Anima Mundi), and a creative faculty that transforms experience into art. The "hundred-year-old memory" ((044-013)) that Yeats gives to the poor is not nostalgia but a living connection to tradition — what he would later call the "Spiritus Mundi" in _The Second Coming_.

## Origins and Sources

- **Irish oral tradition**: Memory as the vehicle of folk culture — the seanchai's art of preserving stories across generations without writing
- **The Anima Mundi**: Yeats's concept (from _Per Amica Silentia Lunae_, 1918) of a World Soul or collective unconscious where all human memories pool
- **Platonic anamnesis**: The doctrine that learning is actually recollection — the soul remembers truths it knew before incarnation
- **Romantic memory**: Wordsworth's "emotion recollected in tranquility" as the source of poetry

## Symbolic Meaning

1. **Ancestral memory**: "In some dim memory or ancient mood" ((015-501)) — memory connects the individual to a pre-modern, mythological past that persists beneath the surface of modern consciousness.

2. **Memory and landscape**: Memory is "loaded with the memory of days" ((015-552)), embedded in specific places — the Irish landscape is saturated with historical and mythical memory.

3. **Memory as poetic substance**: The poet transforms "this human life" into art by remembering it — but also by "blotting out" memory ((044-368)), achieving distance through the cold eye.

4. **The memory of wings**: "Where wings have memory of wings" ((043-180)) — memory transcends the individual life, operating across incarnations, through what Yeats would call the _Anima Mundi_.

## Development Across Yeats's Career

**Early period**: Memory is primarily mythological — the "memory of days" loaded onto the landscape of Irish legend. Usheen remembers the heroic past as a living presence.

**Middle period**: Memory becomes philosophical. "We hold because our memory is" ((040-086)) — memory is what gives substance to the present. _Per Amica Silentia Lunae_ (1918) develops the concept of the Great Memory that all souls share.

**Late period**: Memory is simultaneously treasured and erased. "Responsibilities" (1914) gives "a hundred-year-old memory to the poor" ((044-013)) while also acknowledging the need to "blot out of my memory this life" ((044-368)). The poet both preserves and transcends the personal past.
