# Index of Events

All biblical and historical events referenced in the Olney Hymns.

::: query
type: event
display: table
fields: [label, date]
field_labels:
  label: Event
  date: Period / Date
sort: label
order: asc
:::

## About This Index

Each event page examines its biblical account, theological significance,
and the ways Newton and Cowper draw upon it in their hymns.
