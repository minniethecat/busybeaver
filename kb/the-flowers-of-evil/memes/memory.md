# Memory

## Definition

In Baudelaire's poetry, **Memory** (*souvenir*, *mémoire*) is a double-edged faculty: it is the vessel that preserves vanished beauty and lost love, yet it is also the source of an exquisite and inescapable torment. The remembered past does not console the present — it indicts it. In the Baudelairean universe, where time devours everything ("Le Temps mange la vie," 010-022), memory is the sole resistance, but it is a resistance that wounds even as it preserves.

## In Les Fleurs du Mal

Memory pervades *Les Fleurs du Mal* as both explicit theme and structural principle. Baudelaire returns obsessively to the paradox that the past, precisely because it is irrecoverable, exerts a stronger hold than the present.

In **La Chevelure** (018), the lover's hair becomes a vessel of memory — a sensory portal that transports the poet to a tropical world of "langoureuses Asies et brûlantes Afriques." The poem opens with a direct invocation of memory as substance:

> Des souvenirs dormant dans cette chevelure… (018-004)

The hair is not merely beautiful in itself; it is saturated with recollection. And in the poem's voluptuous close, memory becomes an intoxication:

> Je hume à longs traits le vin du souvenir. (018-035)

Memory is a wine — an altered state that blurs the boundary between past and present, real and imagined. The sensory trigger (hair, perfume, sound) is the Baudelairean mechanism for involuntary memory, anticipaing Proust by half a century.

The most concentrated meditation on memory appears in **Le Balcon** (023), where the refrain makes explicit the bargain between memory and pain:

> Mère des souvenirs, maîtresse des maîtresses… (023-001)

The balcony — the remembered scene of erotic evenings — returns each stanza with accumulating force. And the poem's final lines state the bargain plainly:

> Et dont le souvenir pour l'amour me ravive. (023-032)

Memory revives love, but it revives it *as pain* — a fire that illuminates only what is lost.

In **Spleen LXXVI** (054), memory breaks its banks and floods the present:

> J'ai plus de souvenirs que si j'avais mille ans. (054-015)

The hyperbole ("more memories than if I were a thousand years old") is the voice of Spleen at its most absolute — memory as accumulation without meaning, each souvenir a weight added to an already crushing load. The chest of drawers filled with "bilans, vers, billets doux, procès, romances" becomes the mind's own mausoleum.

Yet memory also provides moments of transcendence. In **Le Flacon** (035), a perfume uncorked releases not scent but memory:

> Voilà le souvenir enivrant qui voltige
> Dans l'air troublé… (035-013)

And in **La Cloche Fêlée** (040), memory is identified with bitterness itself:

> Le souvenir cuisant de son limon amer… (040-004)

But even the bitter memory (*souvenir cuisant*) is preferable to the soul's total silence. The cracked bell still rings; the memory still stirs.

## Analysis

Baudelaire's treatment of memory departs from the Romantic tradition in a crucial respect. For the Romantics — Wordsworth in particular — memory was redemptive: the "emotion recollected in tranquility" purified experience into poetry. For Baudelaire, memory is **not redemptive but intensifying**. It does not calm the original emotion; it concentrates it into something more potent and more painful.

This is the logic of Baudelairean memory in four movements: (1) the original experience was intense; (2) time has destroyed it; (3) memory preserves the intensity but not the experience; (4) therefore memory is a wound that does not heal. The Baudelairean self is caught between a past it cannot recover and a present it cannot inhabit — a structure that defines the condition of Spleen itself.

The link to the theory of correspondences is direct. If, as **Correspondances** (004) teaches, the visible world is a "forêt de symboles" (forest of symbols), then memory is the faculty that decodes those symbols — but only after they have vanished. The perfume-souvenir in **Le Flacon** is the exemplary case: a sensory trace that opens onto an entire lost world. Baudelaire here invents, in embryo, the Proustian method: involuntary memory triggered by a physical sensation.

And yet Baudelaire's memory is darker than Proust's. The madeleine restores Combray; Baudelaire's perfume-souvenir restores only the knowledge of loss. This is because in Baudelaire's universe, the object of memory is not a recoverable place but an **irrecoverable intensity** — the lost erotic moment, the vanished beloved, the youth that cannot return. Memory is not the cure for time; it is time's sharpest instrument.
