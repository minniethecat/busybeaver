# Character Index

This page provides a comprehensive index of every named character appearing in *The Mystery at Lilac Inn*, the fourth volume of the Nancy Drew Mystery Stories series. Characters are listed alphabetically with their current quality rating. Click any name to view the full character profile.

## All Characters

::: query
type: person
display: table
fields: [label, quality, total_refs, description]
field_labels:
  label: Character
  quality: Quality
  total_refs: Mentions
  description: Role
sort: label
order: asc
:::

## About This Index

Entries are drawn from the wiki's person-type pages, each built from the original text of the novel (1930). Quality ratings reflect the depth of coverage: **featured** pages include extensive biographical detail and cross-references; **standard** pages meet the baseline for character coverage.
