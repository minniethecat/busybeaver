# Dr. McLaren's Church

## Overview

The progressive church led by Dr. [[doctor-mclaren|McLaren]], where theological discussions about the philosophy take place.

## Description

Part of the setting of Lloyd C. Douglas's *Magnificent Obsession* (1929).

## Narrative Significance

This location serves as a significant setting for key events in the novel, contributing to the atmosphere and meaning of [[bobby-merrick|Bobby Merrick]]'s transformation.

## Key Passages

Described in the novel with vivid detail, this location appears across multiple chapters.
