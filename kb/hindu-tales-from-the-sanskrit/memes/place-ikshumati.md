# Ikshumati

## Description

Ikshumati is a named town in India, briefly mentioned as the setting near which a significant event takes place. It is described as being situated *"on a beautiful wide river"* [084-001], a geographic detail that places it within the riverine landscape characteristic of Indian settlements in the tales.
