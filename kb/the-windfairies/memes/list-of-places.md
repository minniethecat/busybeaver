# List of Places

This page provides a complete, dynamically generated index of all locations and settings
that appear in the tales of *The Windfairies* — from enchanted forests and hills
to the sea, pools, villages, and cottages where the stories unfold.

The index is sorted alphabetically by name.

## All Places

::: query
type: place
display: table
fields: [label, location_type, story, quality, total_refs]
field_labels:
  label: Place
  location_type: Type
  story: Story
  quality: Quality
  total_refs: Citations
sort: label
order: asc
title: All Locations
:::
