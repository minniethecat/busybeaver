# List of Events

This page provides a complete, dynamically generated index of all significant events
and turning points that drive the narratives in *The Windfairies*.
Each entry links to a dedicated page describing the event's context, participants, and outcome.

The index is sorted alphabetically by name.

## All Events

::: query
type: event
display: table
fields: [label, event_type, story, quality, total_refs]
field_labels:
  label: Event
  event_type: Type
  story: Story
  quality: Quality
  total_refs: Citations
sort: label
order: asc
title: All Story Events
:::
