# List of Concepts

This page provides a complete, dynamically generated index of all abstract concepts,
themes, and recurring motifs that weave through the tales of *The Windfairies*.
From love and music to jealousy and transformation, each entry links to a dedicated page
exploring the concept's manifestations across multiple stories.

The index is sorted alphabetically by name.

## All Concepts

::: query
type: concept
display: table
fields: [label, themes, quality, total_refs]
field_labels:
  label: Concept
  themes: Related Themes
  quality: Quality
  total_refs: Citations
sort: label
order: asc
title: All Concepts and Themes
:::
