# List of Characters

This page provides a complete, dynamically generated index of all human characters
who appear in the fairy tales collected in *The Windfairies*.
Each entry links to a dedicated page with the character's role, story, and relationships.

The index is sorted alphabetically by name and includes quality ratings to help
identify the most thoroughly documented entries.

## All Characters

::: query
type: person
display: table
fields: [label, story, quality, total_refs]
field_labels:
  label: Character
  story: Story
  quality: Quality
  total_refs: Citations
sort: label
order: asc
title: All Human Characters
:::
