# Location Index

A gazetteer of every location mentioned in *Perishable Goods*, from English country estates
to Continental way-stations and the Carinthian wilderness.

::: query
type: place
quality: basic
display: table
fields: [label, total_refs]
field_labels:
  label: Location
  total_refs: References
sort: label
order: asc
:::

## About This Index

Locations are listed alphabetically. The "References" column indicates
how often each location is linked from other wiki pages, reflecting its narrative significance.
