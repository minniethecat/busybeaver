# Social Class in Vanity Fair

*Vanity Fair* is among the most acute studies of social class in nineteenth-century English fiction. Thackeray maps a society in flux: old aristocratic families clinging to status, a rising commercial class buying its way into gentility, and a servant class observing it all from below stairs. This page indexes every wiki entry tagged "social-class", gathering the characters, places, and concepts through which the novel anatomises hierarchy.
