# Objects

This page indexes all significant physical objects in *As I Lay Dying*. In Faulkner's world, objects transcend their material function — Addie's coffin, Cash's tools, Jewel's horse — becoming vessels for grief, craft, love, and obsession. Each object entry traces how material things accumulate meaning through the novel's multiple narrative perspectives.

## Notes

Object entries examine the symbolic and narrative functions of material items. The boundary between literal tool and metaphor is deliberately porous in Faulkner's prose — the same object may serve practical function and carry existential weight simultaneously.
