# Characters

This page lists all characters appearing in *As I Lay Dying*, drawn from the wiki's person entries. Each character is rendered by Faulkner through the novel's distinctive multi-narrator structure — fifteen different narrators across fifty-nine chapters — so the same character may appear through radically different perceptual lenses.
