# Characters Index

This page provides a complete, dynamically updated list of all characters
from William Morris's *The [[sundering-flood]]* (1897). Entries are sorted
alphabetically within each quality tier.

## Featured Characters

::: query
type: character
quality: featured
sort: label
order: asc
display: table
fields: [label, quality, total_refs]
field_labels:
  label: Character
  quality: Quality
  total_refs: Mentions
:::
