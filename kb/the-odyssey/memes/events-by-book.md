# Events by Book

This page lists all event entries organised by the book of the *Odyssey* in which each event takes place. Events spanning multiple books are listed under their primary book reference (the first book in which the event's main action occurs).

::: query
type: event
sort: book
order: asc
display: table
fields: [label, book, description, quality]
field_labels:
  label: Event
  book: Book
  description: Summary
  quality: Quality
title: All Events (Chronological by Book)
:::

The *Odyssey* comprises 24 books, each containing key narrative events. This index provides a structural overview of the epic's major episodes, useful for readers navigating the poem's non-linear narrative structure (in which Books 9–12 form an extended flashback narrated by Odysseus himself).
