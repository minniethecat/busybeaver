# Character Index

This page provides a searchable, auto-updating index of all characters appearing in
*Brewster's Millions* by George Barr McCutcheon (1902).

::: query
type: person
display: table
fields: [label, role, affiliation, total_refs, quality]
field_labels:
  label: Character
  role: Role
  affiliation: Affiliation
  total_refs: References
  quality: Quality
sort: label
order: asc
title: All Characters
:::

## About This Index

This index is dynamically generated from wiki page frontmatter. Characters are listed
alphabetically by display name. The "References" column shows how many wiki pages link
to each character, serving as a rough popularity/influence metric.

Characters marked as `featured` have been identified as high-centrality or
high-search-traffic entities deserving of premium page quality.
