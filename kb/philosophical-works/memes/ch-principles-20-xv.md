(073-001) That necessary existence is not in the same way comprised in the notions which we have of other things, but merely contingent existence.

(073-002) The mind will be still more certain of the truth of this conclusion, if it consider that it has no idea of any other thing in which it can discover that necessary existence is contained; for, from this circumstance alone, it will discern that the idea of an all-[[perfection|perfect]] Being has not been framed by itself, and that it does not represent a chimera, but a true and immutable nature, which must exist since it can only be conceived as necessarily existing.
