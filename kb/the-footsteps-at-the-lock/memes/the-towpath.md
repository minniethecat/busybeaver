# The Towpath

The riverside path from Shipcote Lock toward Eaton Bridge that Nigel claims to have run along, chasing the drifting canoe (002-008).

## Description

Uneven, sometimes muddy, interrupted by vegetation. Running at speed would be difficult.

## Role in the Story

Bredon walks the towpath himself, timing the distance. The path's physical reality provides a check on Nigel's verbal account. It is the thread connecting the novel's key locations: lock, bridge, island, inn (004-043).
