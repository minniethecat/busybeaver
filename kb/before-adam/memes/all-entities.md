A single alphabetical listing of every documented entity in the *Before Adam* wiki. Unlike the type-specific indexes — which group entries by category and sort by quality — this directory lets you find any entry by name. Use it as a site map or quick reference.

::: query
display: table
fields: [label, type, quality, total_refs]
field_labels:
  label: Entity
  type: Type
  quality: Quality
  total_refs: References
sort: label
order: asc
title: All Entities
:::

## Reading Guide

The **Type** column identifies each entry's category — Person (characters), Place (locations), Event (narrative turning points), Concept (ideas and themes), Group (tribes and clans), or Species (animals). The **Quality** column reflects development depth. Use the type-specific indexes ([[person-index|People]], [[place-index|Places]], [[event-index|Events]], [[concept-index|Concepts]], [[group-index|Groups]], [[species-index|Species]]) for focused browsing within each category.
