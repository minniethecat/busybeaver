# The British Class System

## Overview

The British [[social-class|class]] system is the meta-institution that governs all the novel's social dynamics. It determines who can marry whom, which professions confer status, and how individuals value themselves and others (008-001). The novel critiques it implicitly — through [[kitty-fane|Kitty]]'s realization that the values it inculcates are hollow — while acknowledging its inescapable power (008-006).
